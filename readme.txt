# A*算法在随机生成迷宫中的应用

## 概述

这个Python脚本演示了A*路径规划算法应用在一个随机生成的迷宫中。迷宫被表示成一个图，其中节点是迷宫中的位置，边表示节点之间的可能路径。

## 依赖

- Python 3.x
- NumPy
- Matplotlib

使用以下命令安装所需的依赖：

pip install numpy matplotlib


## 使用方法

1. 在Python环境中运行脚本（`astar_maze.py`）。
2. 在提示时输入迷宫的尺寸（长和宽）。
3. 脚本将生成一个随机迷宫，应用A*算法找到最短路径，并使用Matplotlib可视化整个过程。

## 脚本结构

- `map.py`: 定义了表示迷宫图和可视化的`Map`类。
- `astar.py`: 使用`AStar`类实现A*算法。
- `astar_maze.py`: 用户交互、迷宫生成和可视化的主要脚本。

## 可视化

脚本使用Matplotlib可视化迷宫生成、A*搜索过程和最终最短路径。颜色代表算法中的不同阶段：
- 黄色：迷宫边
- 蓝色：未探索的节点
- 橙色：起点和终点节点
- 浅蓝色：搜索过程中探索的节点
- 浅橙色：最短路径

## 示例

请自行定义迷宫的长和宽：
5 5
